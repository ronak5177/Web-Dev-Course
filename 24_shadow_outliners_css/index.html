I am a box

Syntax

box-shadow: h-offset v-offset blur spread color inset; text-shadow: h-offset v-offset blur color; outline: width style color;

Box Shadows

  1. h-offset and v-offset: Determines the shadow's horizontal and vertical position
  2. blur: The larger the value, the blurrier the shadow.
  3. spread: Expands or shrinks the shadow size.
  4. color: Defines the shadow color.
  5. inset: Makes the shadow inner.
